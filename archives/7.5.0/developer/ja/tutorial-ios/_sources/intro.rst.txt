===========
はじめに
===========

.. include:: ../common/linkdef.rst

本マニュアルでは、iOS ネイティブアプリケーション向けの
チュートリアルを提示します。

チュートリアルのソースコード
============================

チュートリアルのソースコード一式は Github で配布していますので、
こちらから入手してください。

URL は https://github.com/nec-baas/baas-tutorials-ios です。

ビルドに必要な環境
==================

iOS の開発環境が必要になります。具体的には Mac および Xcode が必要となります。
詳細は `Apple Developer`_ を参照してください。

また、CocoaPods の導入も必要です。手順は `CocoaPods`_ を参照してください。

Xcode でプロジェクトを開く手順
===============================

Pod のインストール
------------------

プロジェクトを開く前に、各プロジェクトディレクトリ内で以下コマンドを実行し、
BaaS iOS SDK ライブラリの Pod をインストールしてください。

.. code-block:: shell

    $ pod install

インストール後、.xcworkspace が作成されるはずです。

プロジェクトを開く
------------------

Xcode を起動し、File → Open を選択し、各チュートリアルのディレクトリを指定して
プロジェクトを開いてください。

または、各チュートリアルにある \*.xcworkspace ファイルをダブルクリックして開くこともできます。

開発用のプロビジョニングプロファイルを適切に設定したうえで、ビルド・実行を行ってください。
なお、App ID は各プロジェクトの設定を確認してください。
