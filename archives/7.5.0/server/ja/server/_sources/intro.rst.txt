本文書の位置づけ
================

本文書は、モバイルバックエンド基盤サーバ(以下 BaaS サーバ)の運用者向けのドキュメントです。
本文書では、以下の内容について説明します。

* BaaS サーバのインストール手順
* BaaS サーバのコンフィグレーション手順
* BaaS サーバに関連するサーバ (DB, リバースプロキシ, など)のセットアップ手順

BaaS サーバについて
=====================

BaaS サーバは MBaaS (Mobile Backend as a Service) 機能を提供するサーバです。
以下の機能を提供します。

* ユーザ管理・認証
* グループ管理
* オブジェクト(JSON)ストレージ
* ファイルストレージ
* API Gateway / Cloud Functions
* Push通知
* システム連携

BaaS サーバは Java で記述されており、Tomcat などのアプリケーションサーバ
上で動作します。データベースとしては MongoDB を使用します。
