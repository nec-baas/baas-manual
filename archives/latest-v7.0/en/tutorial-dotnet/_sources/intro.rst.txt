はじめに
===========

.. include:: ../common/linkdef.rst

本マニュアルでは、.NET アプリケーション向けのチュートリアルについて記載します。

チュートリアルのソースコード
============================

チュートリアルのソースコード一式を別途配布しています。

ビルドに必要な環境
==================

Visual Studio 2013 Professional 以上が必要になります。

詳細は `Visual Studio`_ のサイトを参照してください。

Visual Studio へのインポート手順
=================================

Visual Studio を起動し、ファイル → 開く → プロジェクト/ソリューションを選択し、各チュートリアルのソリューションファイル (.slnファイル) を指定してください。
