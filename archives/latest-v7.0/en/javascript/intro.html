1. はじめに¶

本文書は、モバイルバックエンド基盤 Javascript SDK の開発ガイドです。

本文書では、SDK のセットアップ手順と SDK 各機能の利用方法について解説します。

1.1. 動作環境¶

JavaScript SDK の動作環境に関しては、 マニュアル ⇒ 「動作環境」を参照してください。

1.2. 注意事項¶

IE11 ブラウザ上で本 SDK を動作させる場合は以下の点にご注意ください。

  • IE11 では、ローカルファイルシステムからコンテンツ( JavaScript )を読み込んだ場合、ログイン処理などの一部 API の実行に失敗します。(ログ出力上は、 CORS 関連でエラーが発生したように見える場合があります)

    これは、IE11 でローカルコンテンツを読み込んで実行する際、 HTML5 の Web Storage（localStorage） が 動作しないことが原因です。（ IE11 のセキュリティ制限により locarStorage 関連の API 実行時にエラーとなる)

    コンテンツを Web サーバ上から取得する場合には、本問題は発生しません。

    なお、本問題は IE11 でのみ発生し、Chrome等の他ブラウザでは発生しません。(2015.12月時点)

  • イントラネット上のコンテンツサーバに IE11 でアクセスした時、うまく動作せず、Chrome 等の他ブラウザで問題がない場合は、IE11 にて以下設定を行ってください。

    「ツール」→「互換表示設定」→「イントラネットサイトを互換表示で表示する」のチェックを外す。
