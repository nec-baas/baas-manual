==============
API定義の登録
==============

Swagger の API 定義ファイルの作成が完了したら、API 定義をサーバに登録します。

CLI ツールから投入する場合
--------------------------

.. code-block:: bash

    $ nebula create-api [定義ファイル名]

APIの一覧は "nebula list-apis" で確認できます。

APIを削除したい場合は "nebula delete-api <API名>" としてください。

デベロッパーコンソールから投入する場合
--------------------------------------

デベロッパーコンソールにログインし、画面左メニューの「API Gateway」を選択してください。

「追加」をクリックすると、API Gateway Swagger 編集画面に遷移します。
API名を設定し、API定義に Swagger 定義ファイルの内容をコピー・ペーストして「保存」します。
