====================
Cloud Functions
====================

Function 表示・編集とログ表示ができます。

Function 表示・編集
======================

API Gateway と同様、Function の定義情報の一覧表示・詳細表示・追加・編集ができます。

Function 定義情報は Swagger の YAML 形式テキストで表示・入力をサポートします。


ログ表示
======================

Cloud Functions のログ表示ができます。以下の条件でログの絞込ができます。

* 開始日時
* 日時指定・現在時刻のn分前で指定
* 最大表示ログ数
* Function 名
* ログレベル

.. attention::

  ログ表示は、fluentd を導入して MongoDB にログ収集を行っている場合のみ可能です。
