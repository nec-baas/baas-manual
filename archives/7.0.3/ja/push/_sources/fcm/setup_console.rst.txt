=======================================
NEC BaaS サーバへの API キーの設定
=======================================

アプリケーションの作成
======================

APIキーが取得できたら、MBaaS サーバに APIキーを登録します。
MBaaS デベロッパコンソールにログインし、アプリケーションを作成してください。

APIキーの登録
==============

デベロッパコンソールのアプリケーションの編集画面に移動してください。

.. image:: ../images/Console1.png

「FCM設定」の「サーバーキー」欄にFCM のサーバーキーを入力してください。
(旧 GCM を使用する場合は GCM キーを入力してください)

また、クライアントからの Push 送信を許可する場合、「クライアントPush送信許可」
にチェックを入れる必要があります。
