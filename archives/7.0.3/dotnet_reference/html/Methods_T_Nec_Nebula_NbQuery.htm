Projection
プロジェクションを設定する。
取得したいフィールド名を列挙する。
フィールドを抑制したい場合は、フィールド名の先頭に "-" を付与する。
列挙・抑制を混在させることはできない。例外として、_id のみを抑制することは可能。
Copy
// 例1) name のみを含める場合
query.Projection("name");

// 例2) address のみを除外する場合
query.Projection("-address");

// 例3) name を含め、_id を除外する場合
query.Projection("name", "-_id");
