4.1. Node.js プロジェクトのセットアップ¶

4.1.1. サンプルプロジェクト¶

サンプルプロジェクトは、Github で配布しています。

また、CLI の examples ディレクトリにもサンプルが入っています。

4.1.2. package.json ファイルの作成¶

プロジェクトディレクトリ内で以下のコマンドを実行し、package.json ファイルを作成してください。

$ npm init
