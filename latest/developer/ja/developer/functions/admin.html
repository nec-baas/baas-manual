2.13. 管理者向け機能¶

管理者向け機能は、下記のような管理者向けの操作を提供します。

  • テナント操作
  • アプリケーション操作
  • 管理者操作
  • テナント所属管理者操作
  • ライセンスキー操作
  • システム設定操作 など

詳細については、「 REST API リファレンス: 管理API編 」 をご覧ください。
