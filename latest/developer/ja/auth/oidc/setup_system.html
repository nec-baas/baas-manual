6.6. システムの事前設定¶

OpenID Connect 認証の場合、事前にシステム設定が必要となります。

システム設定は、デベロッパーコンソール画面で行います。詳細は、システム設定 を参照してください。

6.6.1. 一般設定 - APIサーバ ベースURI¶

OpenID Connectプロバイダ (OP) に設定するリダイレクト URI にて使用されるため、設定が必要となります。

注意

IPアドレスではなく、FQDN を設定してください。Google などの一部の OP では、リダイレクト URI に IPアドレスを指定できないためです。

6.6.2. プロキシサーバ設定¶

プロキシサーバを経由して OP に接続する場合、プロキシサーバホスト、プロキシサーバポート番号の設定が必要となります。

注意

プロキシサーバホスト設定状態で、プロキシサーバを経由せずに OP に接続する場合は、プロキシ例外リストにプロキシ経由外とするホスト名を設定してください。
