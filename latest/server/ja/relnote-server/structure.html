リリース物件の構成¶

本リリースには以下の物件が収録されています。 なお、各サーバのバージョンの詳細は バージョン対応表 を参照してください。

  • NECモバイルバックエンド基盤 サーバ

    InstallMedia/
    ├── baas-server-*.tar.gz
    ├── cloudfn-servers-*.tar.gz
    ├── ssepush-server-*.tar.gz
    └── readme.txt
    • baas-server-*.tar.gz : API サーバ、コンソールサーバ 配布物
    • cloudfn-servers-*.tar.gz : Cloud Functions サーバ 配布物
    • ssepush-server-*.tar.gz : SSE Push サーバ 配布物
    • readme.txt ： 配布物の説明

注意

リリースによっては、コンポーネントの一部のみのリリースや特定のお客様向けのリリースとなっている場合がございますのでご注意ください。

SDK はリリース物件には含まれません。別途提供となります。
