==================
リリース物件の構成
==================

本リリースには以下の物件が含まれます。

なお、本リリースに対応する各 SDK のバージョン番号の詳細は `バージョン対応表 <../../version_info.html>`_ を参照してください。

* SDK (※1)

  - JavaScript SDK
  - Java / Android SDK
  - .NET SDK
  - iOS SDK (※2)
  - Embedded SDK
  - Cloud Functions SDK
  - Python SDK
* デベロッパーマニュアル

| ※1: 一部の API はβ版提供となります。詳細は `注意事項 <../notes-sdks/index.html>`_ を参照してください。
| ※2: β版提供です。
