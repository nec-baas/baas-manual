11.2. ヘルスチェック¶

GET /1/_health¶

BaaSサーバの稼働状態を返す

Status Codes:
  • 200 OK -- 正常取得
Response JSON Object:
 
  • name (string) -- サーバ名("api"固定)
  • state (string) -- 稼働状態("running"固定)

レスポンスボディの例

{
    "name": "api",
    "state": "running"
}
