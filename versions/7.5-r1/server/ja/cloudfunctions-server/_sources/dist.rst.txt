配布物の構成
------------

Cloud Functions サーバ配布物の構成は以下のようになっています。

* cloudfn-server-manager-$(version).tar.gz : サーバマネージャ
* cloudfn-node-server-$(version).tgz : Node ロジックサーバ
* cloudfn-java-server-$(version).tgz : Java ロジックサーバ
* cloudfn-docker-$(version).tar.gz : Docker 関連ファイル
* install.sh : インストーラ

以下の説明では、配布物を格納したディレクトリを $(DISTDIR) として説明します。
