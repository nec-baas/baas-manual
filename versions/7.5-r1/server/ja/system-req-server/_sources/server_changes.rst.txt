==========================
サーバ: v7.0 との主な差異
==========================

モバイルバックエンド基盤 ver 7.0 とは以下の差異があります。

全サーバ
========

* Java

  - v7.0: OpenJDK8 または Oracle JDK 8
  - v7.5: OpenJDK8, OpenJDK11 または Oracle JDK 8

* Tomcat

  - v7.0: 8.0.50 以上
  - v7.5: 8.5.32 以上

* MongoDB

  - v7.0: 3.2 以上
  - v7.5: 3.6 以上 (4.0以上推奨)

* RabbitMQ

  - v7.0: 3.6.1 以上
  - v7.5: 3.7 以上
