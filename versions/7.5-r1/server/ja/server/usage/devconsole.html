3.6. デベロッパーコンソール利用手順¶

ここでは、デベロッパーコンソールのログイン用アカウントと テナント・アプリケーションを作成する手順について説明します。

なお、画面イメージを用いた詳細な説明は デベロッパーコンソール利用ガイド を参照してください。

3.6.1. 初回ログイン¶

ブラウザからデベロッパーコンソール(http://[hostname:port]/console/) にアクセスし、以下の初期システム管理者アカウントでログインしてください。

  • ユーザ名: admin@example.com
  • パスワード: admin

ログインが完了した時点でパスワード設定画面が表示されますので、 パスワードを設定してください。

注意

CLIツールなどですでにデベロッパーを1名以上追加済みの場合、初期システム管理者アカウントは 自動追加されません。

注釈

ログイン時に CSRF Token エラーがでた場合は、画面をリロードして再度ログインしてください。

3.6.2. テナントの作成¶

「管理」⇒「テナント管理」⇒「追加」より、テナントを作成してください。

テナントが正常に作成されると、画面にテナントIDが表示されます。 この値は REST API 呼び出し時に使用します。

3.6.3. テナント内にアプリケーションを作成¶

テナントを選択し、「アプリケーション」⇒「追加」と進みます。 アプリ名と説明文を入力して追加を押すと、アプリケーションが追加されます。

作成されたアプリを選択すると、アプリケーションID、アプリケーションキー、マスターキーを確認することができます。
