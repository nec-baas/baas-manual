4. ログ出力手順¶

ログ出力を行う手順について説明します。

4.1. ログ出力設定¶

  • JavaScript SDK の Nebula クラスにある初期化メソッド「initialize」で初期化パラメータの debugMode に「debug」を指定します。この設定により、ブラウザ(Chrome等)のデバッグコンソール上にログが出力されます。
    • JavaScript SDK のチュートリアル4では、設定ファイル config.js の debugMode に「debug」を指定し、この値を上記「initialize」に読み込ませています。
    • debugMode に「release」（デフォルト）を指定した場合は、クリティカルなエラー以外のログは出力されません。

4.2. ログ出力手順¶

  • ブラウザ(Chrome等)のデバッグコンソール上にログが出力されます。

4.3. JavaScript アプリでのログ出力¶

  • JavaScript アプリ本体のログ出力を行う場合も前述の内容と手順は同じです。以下のような出力関数を JavaScript アプリのソースコード上に記述することで該当個所がログに出力されます。
console.log("Hello world");
