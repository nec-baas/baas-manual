3.7. システムログ¶

システム管理者は、システムログから、モバイルバックエンド基盤の出力するログを参照することができます。

「管理」の「システムログ」を選択します。

以下条件で、ログの絞り込みをすることができます。

  • 日時
  • 件数
  • テナントID
  • アプリケーションID

またログの出力レベルによる絞り込みにも対応しています。

  • ALL
  • ERROR
  • WARNING
  • INFO
  • DEBUG
  • TRACE

注意

システムログ表示は、fluentd を導入して MongoDB にログ収集を行っている場合のみ可能です。
