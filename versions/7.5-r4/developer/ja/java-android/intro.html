1. はじめに¶

本文書は、モバイルバックエンド基盤 Java / Android SDK の開発ガイドです。

本文書では、SDK のセットアップ手順と SDK 各機能の利用方法について解説します。

1.1. 動作環境¶

Java / AndroidSDK の動作環境に関しては、 マニュアル ⇒ 「動作環境」を参照してください。

Java / Android SDK 7.5.x は BaaS サーバ 7.5 以上に対応しています。

注意

BaaSサーバ 7.5 未満には対応していません。
