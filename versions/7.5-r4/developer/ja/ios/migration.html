11. マイグレーションガイド¶

本章では、NEC BaaS iOS SDK の以前のバージョンからの API の変更点や移行方法について説明します。

11.1. ver 5.0.0 → 7.5.0 での変更点¶

CocoaPods に対応しました。

11.2. ver 3.0.0 → 5.0.0 での変更点¶

11.2.1. Swift3 / Xcode8 対応¶

Swift3 / Xcode8 に対応しました。

詳細は Swift の利用 を参照してください。

11.2.2. 一部の enum / options 名称変更¶

一部の enum の名称を単数形に変更しました。

  • NBAclPermissions -> NBAclPermission
  • NBErrorCodes -> NBErrorCode
  • NBFileConditions -> NBFileCondition
