9.3. テナント所属管理者削除¶

DELETE /1/_sysadm/(tenantId)/tenantDevelopers/(developerId)¶

テナントから管理者を削除する

Request Headers:
 
  • X-Developer-Token -- 管理者用セッショントークン（必須）
Parameters:
  • tenantId (string) -- テナントID
  • developerId (string) -- 管理者ID
Status Codes:
  • 200 OK -- 正常に削除した
  • 401 Unauthorized -- 認証エラー
  • 403 Forbidden -- 権限エラー
  • 404 Not Found -- 該当テナントもしくは管理者ユーザが存在しない、またはテナントが無効
Response JSON Object:
 
  • result (string) -- 実行結果

レスポンス

リクエスト成功時は以下。

{
  "result":"ok"
}

注意事項

  • システム管理者またはテナント管理者権限が必要。
