2. 表記について¶

'(' と ')' で囲まれた表記は、それぞれ以下のように読み替えること。

  • (developerId) : 管理者ユーザ ID。管理ユーザ毎に割り当てられるユニークな ID。
    • 管理者ユーザ IDは、MongoDBの ObjectID の文字列表記である。これは 16 進数表記 24 文字の文字列である。
    • 例）"5a93e8005bd67428ee3fac08"

その他の表記については、REST API リファレンス 本編 を参照のこと。
