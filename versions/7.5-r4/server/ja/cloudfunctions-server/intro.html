1. イントロダクション¶

1.1. 本文書の位置づけ¶

本文書は、Cloud Functionsサーバの運用者向けのドキュメントです。 本文書では、以下の内容について説明します。

  • Cloud Functionsサーバのインストール手順
  • Cloud Functionsサーバのコンフィグレーション手順

1.2. Cloud Functionsサーバについて¶

Cloud Functionsサーバは、利用者が作成した任意のコードをサーバ上で実行する実行基盤です。

利用者は JavaScript / Java で記述した任意の Cloud Functions を、モバイルバックエンド基盤 サーバ上の Node.js / Java 環境で実行することができます。

モバイルバックエンド基盤サーバの API ゲートウェイと組み合わせることで、 カスタム REST API を実現することができます。
