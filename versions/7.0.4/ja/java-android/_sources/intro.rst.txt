===================
はじめに
===================

本文書は、モバイルバックエンド基盤 Java / Android SDK の開発ガイドです。

本文書では、SDK のセットアップ手順と SDK 各機能の利用方法について解説します。

動作環境
================

Java / AndroidSDK の動作環境に関しては、 `マニュアル <../../index.html>`__ ⇒ 「動作環境」を参照してください。

Java / Android SDK 7.0.x は BaaS サーバ 7.0 以上に対応しています。

.. attention::

  BaaSサーバ 7.0 未満には対応していません。
