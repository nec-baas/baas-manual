========
認証方式
========

APNs には以下２通りの認証方式があります。
APNs を利用する前に、いずれかの方式を選択し、必要なセットアップを実施する必要があります。

* TLS認証: X.509証明書を用いたTLS認証です。
* トークン認証: JWT(JSON Web Token)を用いた認証です。

トークン認証のほうが新しい方式で、現在はトークン認証のほうが推奨されています。

なお、TLS認証を使用する場合、X.509証明書の有効期限が1年のため、毎年証明書を更新する必要があります。
トークン認証を使用する場合は、鍵の有効期限はありませんので、更新の手間がかかりません。
