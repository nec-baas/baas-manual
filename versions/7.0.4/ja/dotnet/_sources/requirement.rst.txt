===================
動作環境・必要環境
===================

ターゲット環境
==============

.NET SDK のターゲット環境は以下の通りです。

* デスクトップアプリケーション

  - Windows 7, 8.1, 10, Windows Server 2012 R2
  - .NET Framework 4.5.2 以上

* Windows ストアアプリケーション

  - Windows 8.1 (注: オフライン機能は使用不可)
    
* Windows Phone

  - Windows Phone 8.1 (注: オフライン機能は使用不可)

.NET SDK 7.0.x は BaaS サーバ 7.0 以上に対応しています。

.. attention::

  BaaSサーバ 7.0 未満には対応していません。


開発に必要な環境
================

* Visual Studio 2013 Professional 以上
