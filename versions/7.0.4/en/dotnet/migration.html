14. マイグレーションガイド¶

本章では、.NET SDK の以前のバージョンからの API の変更点や移行方法について説明します。

14.1. ver 6.5.0 → 7.0.0 での変更点¶

14.1.1. API Gateway 実行用のAPI追加¶

API Gateway向けの呼び出しを行うための NbApiGatewayクラスを追加しました。 詳細は、API Gateway を参照下さい。

14.1.2. Aggregation 実行用のAPI追加¶

NbObjectBucketにAggregation(集計)実行用のAPI AggregateAsync() を追加しました。 詳細は、 オブジェクトの集計 を参照下さい。

14.2. ver 4.0.1 → 6.5.0 での変更点¶

14.2.1. SAML/OpenID Connect 認証用の API (ワンタイムトークンログイン) の追加¶

ログインAPI NbUser.LoginAsync() を追加しました。 詳細は、ワンタイムトークンログイン を参照してください。

14.3. ver 4.0.0 → 4.0.1 での変更点¶

14.3.1. RESTリクエストタイムアウト・キャンセルAPIの追加¶

HTTP REST リクエストのタイムアウト設定、およびリクエストキャンセルAPIを追加しました。 詳細は、HTTPリクエストタイムアウト・キャンセル を参照してください。
