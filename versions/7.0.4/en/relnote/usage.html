マニュアルの読み方¶

NECモバイルバックエンド基盤の概要を知るには¶

NECモバイルバックエンド基盤の機能・概要については、マニュアルに含まれる、 「 NECモバイルバックエンド基盤入門 」に記載しています。 最初はまずこの文書からお読みになることをおすすめします。

より詳細について知りたい場合は、「 デベロッパーマニュアル 」をご覧ください。

クライアントアプリケーションの開発¶

クライアントアプリケーションを開発するには、各 SDK 毎に用意されている 開発ガイド・SDKマニュアル をお読みください。

サーバの構築¶

モバイルバックエンド基盤の各サーバを構築する手順については、 サーバマニュアル を参照してください。

  • モバイルバックエンド基盤の「APIサーバ・デベロッパーコンソール」を構築するにあたっては、 モバイルバックエンド基盤 サーバ利用手順書 を参照してください。
  • 「SSE Pushサーバ」を構築するにあたっては、「 SSE Push サーバ利用手順書 」を参照してください。
  • 「Cloud Functions サーバ」を構築するにあたっては、「 Cloud Functions サーバ利用手順書 」を参照してください。
