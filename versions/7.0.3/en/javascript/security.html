13. セキュリティ¶

13.1. CORS (Cross Origin Resource Sharing)¶

ブラウザ上で JavaScript SDK を使用する場合、CORS を使用して BaaS API 呼び出し時についてクロスサイトアクセスを許可・制限することができます。

CORS の詳細についてはデベロッパマニュアルの 「Cross Origin Resource Sharging」 を参照してください。
