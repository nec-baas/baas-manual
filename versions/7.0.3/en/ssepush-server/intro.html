  • SSE Push サーバのインストール手順
  • SSE Push サーバのコンフィグレーション手順

1.2. SSE Push サーバについて¶

SSE Push サーバは、HTTP の SSE (Server Sent Event) 仕様に基づく Push 配信サーバです。 Server Sent Event は、サーバから任意のタイミングでクライアントに通知を送信することができるプロトコルです。

iOS APNs や Android GCM/FCM とは異なり、独自に設置した Push サーバからクライアントに Push 配信を行うことができます。 このため、閉域網内で Push 配信を行うことが可能です。

SSE Push サーバはモバイルバックエンド基盤サーバと共に運用することを想定しています。 (SSE Push サーバのみを単独で運用することも可能ですが、この場合は別途フロントエンドサーバを開発する必要があります)
