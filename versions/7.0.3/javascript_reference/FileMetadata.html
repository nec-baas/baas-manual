getFileETag()

fileETag(ファイルに対するETag)を取得する.

fileEtagがセットされていない場合は null を返す
同期が完了していないファイルは、同期されるまで null が設定されている
fileETag値は、サーバ側で同期するときに使用され、一致しない場合は衝突と判断される

Returns:

fileEtagを返す。セットされていない場合は、null を返す。

String
