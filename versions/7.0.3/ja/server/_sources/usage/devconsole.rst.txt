===============================
デベロッパーコンソール利用手順
===============================

ここでは、デベロッパーコンソールのログイン用アカウントと
テナント・アプリケーションを作成する手順について説明します。

なお、画面イメージを用いた詳細な説明は
`デベロッパーコンソール利用ガイド <../../devconsole/index.html>`_
を参照してください。

.. _initial_login:

初回ログイン
============

ブラウザからデベロッパーコンソール(http://[hostname]/console/)
にアクセスし、以下の初期システム管理者アカウントでログインしてください。

* ユーザ名: admin@example.com
* パスワード: admin

ログインが完了した時点でパスワード設定画面が表示されますので、
パスワードを設定してください。

.. attention::

  CLIツールなどですでにデベロッパーを1名以上追加済みの場合、初期システム管理者アカウントは
  自動追加されません。

.. note::

  ログイン時に CSRF Token エラーがでた場合は、画面をリロードして再度ログインしてください。

テナントの作成
===============

「管理」⇒「テナント管理」⇒「追加」より、テナントを作成してください。

テナントが正常に作成されると、画面にテナントIDが表示されます。
この値は REST API 呼び出し時に使用します。

テナント内にアプリケーションを作成
===================================

テナントを選択し、「アプリケーション」⇒「追加」と進みます。
アプリ名と説明文を入力して追加を押すと、アプリケーションが追加されます。

作成されたアプリを選択すると、アプリケーションID、アプリケーションキー、マスターキーを確認することができます。
