1. はじめに・前提条件¶

本文書は、NECモバイルバックエンド基盤サーバ(以下 BaaS サーバ)の諸元を説明したものです。

1.1. 前提条件¶

本文書の諸元値は以下の想定に基づいて記載されています。

  • サーバ: AWS環境を使用。AWS のインスタンスタイプは m3.medium。OS は CentOS 7/64bit。

1.2. 注意事項¶

本文書に記載する諸元値は、システムテストを行い動作を確認した際の設定値(最低保証値)となります。 このため、実際の制限値や性能の実力値よりは低めの数値となっています。

また、ここに記載した設定値で上限値を設定しているわけではありません。

例えば、JSONオブジェクトストレージにおいて1バケット当たりの最大オブジェクト数が 1,000,000 となっていますが、 実際にはストレージサイズが許す限り無制限にデータを格納することが可能です(ほぼ、MongoDB の性能に依存)
