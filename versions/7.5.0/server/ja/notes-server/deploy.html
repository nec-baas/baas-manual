1. デプロイ¶

1.1. ホットデプロイ時の注意¶

Tomcat 上に WAR ファイルをホットデプロイした際は、必ず Tomcat を再起動するようにしてください。

再起動しなくても動作上は大きな問題はありませんが、ホットデプロイを繰り返すと一部のクラスのメモリが解放されずメモリ消費量が上昇したままになる場合があります。
