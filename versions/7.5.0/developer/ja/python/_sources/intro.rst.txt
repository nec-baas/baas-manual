=========
はじめに
=========

本文書は、モバイルバックエンド基盤 Python SDK の開発ガイドです。

本文書では、SDK のセットアップ手順と SDK 各機能の利用方法について解説します。

動作環境
=========

Python 3 が必要です。

詳細な動作環境に関しては、 `動作環境 <../system-req-sdks/index.html>`_ を参照してください。

.. attention::

   Python 2.7 でも動作しますが、正式動作保証対象外とさせていただきます。

Python SDK 7.5.x は BaaS サーバ 7.5 以上に対応しています。

.. attention::

  BaaSサーバ 7.5 未満には対応していません。
