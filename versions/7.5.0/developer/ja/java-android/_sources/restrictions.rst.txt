===================
制限事項
===================

同時コネクション数・スレッド数制限
==================================

本 SDK からサーバに対して発行する非同期API発行時のスレッド数について、以下の制限があります。

* 最大スレッド数: 100 (Pure Java のみ。Android 利用時を除く。)

これらの制限値を変更する場合は SDK のカスタマイズが必要になります。
カスタマイズが必要な場合は、開発元までお問い合わせください。

.. note::

   v6.2.0 より、最大コネクション数、1宛先当たりの最大コネクション数の制約は廃止となりました。
