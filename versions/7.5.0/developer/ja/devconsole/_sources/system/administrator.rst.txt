===========
管理者管理
===========

.. include:: ../../common/linkdef.rst

管理者を追加する場合、「管理」の「管理者管理」から、「追加」を選択します。

.. image:: ../images/devconsole/systemAdmin_1.png
      :scale: 80 %

メールアドレス、ユーザ名、パスワードを設定します。

システム管理者として設定する場合は、チェックボックスを選択してください。選択しない場合は、未割当の管理者として扱います。
:ref:`tenant_developer` から、各テナントの管理者に割り当てることができます。

.. image:: ../images/devconsole/systemAdmin_2.png
      :scale: 80 %

.. attention::

    初回ログイン時のデフォルトユーザは削除し、新たなシステム管理者用のアカウントを作成して下さい。
