4.15. テナント設定 - テンプレート¶

ユーザ管理機能で利用するメールや Web ページのテンプレートを編集することができます。

概要¶

下記の Freemarker テンプレートを編集することができます。

  • メールテンプレート
    • パスワードリセットメール
    • ユーザ登録確認メール
    • ユーザ更新確認メール
    • ユーザ登録・更新完了通知メール
  • Webページテンプレート
    • パスワードリセットページ
    • パスワードリセット完了ページ
    • ユーザ登録・更新確認ページ
    • ユーザ登録・更新確認ページ(パスワード更新付き)
    • ユーザ登録・更新完了ページ

メールテンプレートは件名と本文、 Web ページはページ内容(HTML)を編集できます。

テンプレートパラメータ¶

下記のパラメータを使用することができます。 例えば ${tenantname} と指定することでテンプレートにテナント名を入れることができます。

  • tenantname: テナント名
  • appname: アプリケーション名
  • url: 各テンプレートで使用する URL
  • username: ユーザ名
  • email: メールアドレス
  • status: ユーザの有効/無効状態(Enabled/Disabled)
  • token: パスワードリセットトークン、ユーザ登録・更新確認トークン
  • message: メッセージ文

各テンプレートで使用可能なパラメータは下記です。

使用可能パラメータ

ユーザ更新確認メール

ユーザ登録・更新完了通知メール

email

パスワードリセットページ

ユーザ登録・更新完了ページ

メールテンプレートについては件名、本文の両方に上記パラメータが使用できます。
