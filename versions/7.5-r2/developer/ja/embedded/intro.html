1. はじめに¶

本文書は、NECモバイルバックエンド基盤 Embedded SDK の開発ガイドです。

本文書では、SDK のセットアップ手順と SDK 各機能の利用方法について解説します。

1.1. 動作環境¶

NEC BaaS Embedded SDKはlinux C++ アプリケーション向けのSDKです。 以下プラットフォームをサポートしています。

  • NEC エッジゲートウェイ エントリーモデル
  • Raspberry Pi
  • Open Blocks
  • CentOS 7 (PC)

Embedded SDK 7.5.x は BaaS サーバ 7.5 以上に対応しています。

注意

BaaSサーバ 7.5 未満には対応していません。
