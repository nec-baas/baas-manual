10. Cloud Functions¶

10.1. BaaS API サーバへのアクセスがエラーになります¶

BaaS サーバ側に「APIサーバ内部向けベースURI」の設定がされているか確認ください。 Cloud Functions はこの設定値を元に API サーバの URI を決定しますので、 値が設定されていないと API を呼び出すことができません。

設定値は、デベロッパーコンソールの「システム設定」から変更可能です。
