1. はじめに¶

本文書は、モバイルバックエンド基盤 iOS SDK の開発ガイドです。

本文書では、SDK のセットアップ手順と SDK 各機能の利用方法について解説します。

注意

本 iOS SDK は現時点ではβ品質であり、AS IS で提供するものです。 現時点で品質保証はしておりませんので、ご注意ください。

iOS SDK 7.5.x は BaaS サーバ 7.5 以上に対応しています。

注意

BaaSサーバ 7.5 未満には対応していません。
